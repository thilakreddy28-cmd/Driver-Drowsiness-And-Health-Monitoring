# Driver Drowsiness and Health Monitoring System

## Overview
This project is designed to improve road safety by detecting driver drowsiness and monitoring health conditions in real time.

## Features
- Driver drowsiness detection
- Real-time health monitoring
- Alert generation for unsafe conditions
- Arduino-based implementation

## Technologies Used
- Arduino
- Embedded Systems
- Sensors
- C Programming

## Objective
To reduce road accidents caused by driver fatigue and health-related emergencies.
